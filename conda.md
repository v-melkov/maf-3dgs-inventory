### Установка Miniconda 
1. Скачайте официальный установщик Miniconda для Linux:
```bash
wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
```

2. Запустите процесс установки:
```bash
bash Miniconda3-latest-Linux-x86_64.sh
```

3. Следуйте инструкциям на экране:  

    3.1 Нажимайте **Enter**, чтобы пролистать лицензионное соглашение.  
    3.2 Введите **yes**, чтобы принять условия.  
    3.3 Нажмите **Enter**, чтобы подтвердить стандартный путь установки.  
    3.4 Введите **yes**, когда скрипт предложит инициализировать Conda (conda init)  

4. Перезапустите терминал или примените изменения вручную:
```bash
source ~/.bashrc
```
